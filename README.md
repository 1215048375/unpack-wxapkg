# unpack-wxapkg
Unpack .wxapkg file 

# Usage

```
    [php] /path/to/unpack-wxapkg.php <xxx.wxapkg>

    - Unpack the `xxx.wxapkg` to `xxx.wxapkg.unpacked` directory.
```
